# Ukraine Alarm - OpenClaw Skill

OpenClaw скіл для отримання інформації про повітряні тривоги в Україні через Ukrainian Alarm API.

## Можливості

- 🚨 Перевірка активних тривог по всій Україні
- 📍 Статус конкретного регіону/області/міста
- 📊 Історія останніх 25 тривог для регіону
- 🗺️ Список всіх регіонів України з ID
- 🔄 Перевірка оновлень системи

## Вимоги

- OpenClaw Gateway (версія 1.0+)
- `curl` (зазвичай вже встановлено)
- `jq` (опціонально, для форматування JSON)
- API ключ від ukrainealarm.com (безкоштовний)

## Встановлення

### Крок 1: Отримання API ключа

1. Перейдіть на https://api.ukrainealarm.com
2. Заповніть форму для отримання API ключа
3. Отримайте ключ на ваш email

### Крок 2: Встановлення скіла

#### Варіант А: Автоматичне встановлення

```bash
cd /ukraine-alarm
./install.sh
```

Скрипт автоматично:
- Скопіює скіл в `~/.openclaw/workspace/skills/`
- Створить конфігурацію в `openclaw.json`
- Перевірить залежності

#### Варіант Б: Ручне встановлення

1. Скопіюйте скіл в робочу директорію OpenClaw:

```bash
cp -r ./ukraine-alarm ~/.openclaw/workspace/skills/
```

2. Переконайтеся що curl встановлено:

```bash
which curl
# Має вивести: /usr/bin/curl або подібне
```

3. (Опціонально) Встановіть jq для форматування:

```bash
# macOS
brew install jq

# Linux (Debian/Ubuntu)
sudo apt-get install jq

# Linux (RHEL/CentOS)
sudo yum install jq
```

### Крок 3: Налаштування API ключа

Відредагуйте файл конфігурації OpenClaw:

```bash
nano ~/.openclaw/openclaw.json
```

Додайте або оновіть секцію `skills`:

```json
{
  "skills": {
    "entries": {
      "ukraine-alarm": {
        "enabled": true,
        "env": {
          "UKRAINE_ALARM_API_KEY": "ваш_api_ключ_тут"
        }
      }
    }
  }
}
```

**Важливо:** Замініть `ваш_api_ключ_тут` на реальний API ключ!

### Крок 4: Перезапуск OpenClaw

```bash
# Якщо використовуєте daemon
openclaw daemon restart

# Або просто перезапустіть Gateway
openclaw gateway --port 18789
```

## Перевірка встановлення

## Конфігурація

### Змінні оточення

Скіл використовує наступні змінні оточення:

- `UKRAINE_ALARM_API_KEY` - API ключ (обов'язково)

### Додаткові опції в openclaw.json

```json
{
  "skills": {
    "entries": {
      "ukraine-alarm": {
        "enabled": true,
        "env": {
          "UKRAINE_ALARM_API_KEY": "your_key_here"
        },
        "config": {
          "defaultRegionId": "31",  // За замовчуванням Київ
          "cacheTimeout": 60        // Кеш на 60 секунд
        }
      }
    }
  }
}
```

## Усунення проблем

### Скіл не відображається

1. Перевірте що скіл скопійовано в правильну директорію:
   ```bash
   ls -la ~/.openclaw/workspace/skills/ukraine-alarm/
   ```

2. Перевірте конфігурацію:
   ```bash
   cat ~/.openclaw/openclaw.json | jq '.skills.entries."ukraine-alarm"'
   ```

3. Перезапустіть OpenClaw Gateway

### Помилка автентифікації

1. Перевірте що API ключ встановлено:
   ```bash
   cat ~/.openclaw/openclaw.json | jq -r '.skills.entries."ukraine-alarm".env.UKRAINE_ALARM_API_KEY'
   ```

2. Перевірте валідність ключа на сайті ukrainealarm.com

3. Спробуйте отримати новий API ключ

### Помилка "curl not found"

Встановіть curl:

```bash
# macOS
brew install curl

# Linux (Debian/Ubuntu)
sudo apt-get install curl

# Linux (RHEL/CentOS)
sudo yum install curl
```

### Повільні відповіді

1. Перевірте інтернет з'єднання
2. Перевірте доступність API:
   ```bash
   curl -I https://api.ukrainealarm.com
   ```
3. Можливо API має тимчасові проблеми - спробуйте пізніше
